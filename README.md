## descripcion
Proyecto CRUD de mascotas, productos y paises con seguridad de recursos.

Se empleo: NestJS, TypeScript, Prisma, JWT

## Instalacion

```bash
$ npm install
```

## Ejecucion de aplicacion

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
